A few select works

A selection of works

Troupers - Branding / Web

The project description was to create a website and brand identity to inform customers about a local theatre company as well as giving them the ability to book courses. The client required the website to look fun yet earnest and asked for a minimalist design which was free from clutter. They also requested that all the images must look professional.

I therefore requested that we were able attend one of the acting workshops ourselves to take some photographs using my own equipment. Having been briefed about the branding of the company, I felt that the photographs would be vastly improved if the children were wearing clothes carrying the newly designed logo. I then set out to acquire branded t-shirts for them to wear during the photoshoot which led to creating some fantastic photographs which have been used throughout the website. The company have since reported an increase in course bookings.

Client: Troupers
Date: 11/05/2011

Aknihilation - Viral Flash Game

I am was asked by a pharmaceutical company in Harrogate to construct a viral flash game to promote their range of spot treatment products. They required a game which was fun and engaging and taught the user about the product at the same time. The game features a data capture function which requires users to enter their email address to be given access to the next level of gameplay.

Client: SkniMed
Date: 01/05/2010

Abbey Lodge Studio - Branding / Web

This Leeds based music studio, situated by water, requested an identity based around a warm, homely, log cabin. I designed a logo based on a sketch by the client and built the brand around the log cabin, using wood effects and autumnal colours to represent the cosy feel that was sought after.

The website contains a booking service, that gives bands the opportunity to book through the website and contains a content management system allowing the client to update new features easily. The client also requested flyers, business cards and even coffee mugs as part of the branding package.

Client: Microsoft
Date: 01/05/2011
